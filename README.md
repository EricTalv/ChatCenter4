## Local Setup 

This is how to start the projcet locally:

 0. Have Docker installed and running 

 1. Clone the Repo

 2. CD into project with `Git Bash` (*If On Windows!*)
  
 3. Type -> `./vessel start`

 4. Navigate to -> `localhost`
